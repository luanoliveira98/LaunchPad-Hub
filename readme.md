## ğŸš€ Getting Started

The **LaunchPad Hub** features a built-in interactive CLI to scaffold new projects instantly. This automation ensures that every new service starts with your pre-defined standards for code quality, commit patterns, and architecture.

### 1. Prerequisites

Ensure you have the following installed:

- **Node.js** (v18 or higher)
- One of the following: **npm, yarn, or pnpm**
- **Git**

### 2. Generating a New Template

If you want to create a new blueprint for the Hub:

```Bash
bash scripts/generate-template.sh
```

### 3. Generating a Final Project

To start a new application based on an existing template:

```Bash
bash scripts/generate-project.sh
```

### 3. What happens behind the scenes?

When you run the generator, the script performs the following actions:

- **Clones the Template:** Copies the base structure of the selected framework.
- **Injects Shared Configs:** Copies global configurations (Prettier, ESLint, Commitlint) from the `tooling/` folder.
- **Initializes Git:** Starts a fresh Git repository.
- **Configures Git Hooks:** Installs Husky and sets up the `commit-msg` hook to enforce Conventional Commits.
- **Dynamic Setup:** Renames the project in `package.json` and runs the final installation with your chosen package manager.
